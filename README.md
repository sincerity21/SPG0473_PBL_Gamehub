# GAMEHUB Website
### Instructions:
1. Import database (gamehub.sql)

    - Make new database: 'gamehub'
    - Go to database on PhpMyAdmin, "Import"
    - Choose gamehub.sql
  
2. Download ZIP of project
3. Put on xampp/htdocs/whatever_folder_name_you_want/*

    
**If updating database:**
- Delete all tables under 'gamehub' database as a safe measure
- Re-import updated gamehub.sql
